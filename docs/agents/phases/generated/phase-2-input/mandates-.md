# Mandates - 

⚠️ HARD RULES - These are mandatory by default

## M001: Clean Architecture

**Type:** 

**Description:** All systems must implement 8-layer Clean Architecture

**Rationale:** Foundation enables testability and replaceability

**Status:** `required: true` (Default: mandatory)

**Customizable:** `false` (Hard rules cannot be modified)

**Optional:** `false` (Not negotiable)

**Rationale:** Ensures quality and design clarity

**Customizable:** `false` (Hard rules cannot be modified)
